Tu privacidad es importante

Qué datos se recopilan


El Gobierno Papelero recoge datos estadísticos anónimos sobre tu uso del sitio web
Esto incluye:

  • Sistema operativo
  • Fecha y hora de la visita
  • Dispositivo
  • País
  • Datos sobre el tamaño de tu pantalla
  • Tu idioma preferido
  • Las páginas que visitas dentro de nuestro sitio web
  • Tu navegador
  • El sitio web desde el que se ha originado la consulta


Todos estos datos son anónimos y se mezclan con los de otros usuarios
No recopilamos tu dirección IP

Nota aclaratoria


Este sitio web está también sujeto a la política de privacidad de Github pages y a los términos de servicio de Github pages.
